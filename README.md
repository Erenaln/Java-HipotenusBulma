# 📐 Java Hypotenuse Calculator

**Version:** v1.0 (Console Edition)

A straightforward console application that calculates the hypotenuse of a right-angled triangle using the Pythagorean theorem ($a^2 + b^2 = c^2$). This project was built to practice handling floating-point numbers and utilizing Java's built-in `Math` class.

### ⚙️ Core Mechanics & Concepts
* **Mathematical Operations:** Utilizes `Math.pow()` for squaring the sides and `Math.sqrt()` to find the final length.
* **Precision Handling:** Uses `double` data types to accurately capture and calculate decimal values.
* **User Input:** Captures the lengths of side A and side B dynamically via `java.util.Scanner`.

### 🚀 Upcoming Updates (v2.0)
* **Input Validation:** Adding checks to prevent the user from entering negative values or letters.
* **GUI Implementation:** Upgrading from a console interface to a windowed application using Java Swing.
